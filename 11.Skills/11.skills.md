# Claude Skills 完全指南：让 AI 助手掌握专业技能

## 什么是 Claude Skills？

**Skills** 是 Anthropic 为 Claude 推出的一项革命性功能。简单来说，Skills 是包含指令、脚本和资源的文件夹，Claude 可以动态加载这些内容来提升在特定任务上的表现。

Skills 的核心理念是：**教会 Claude 如何可重复地完成特定任务**。无论是：
- 按照公司品牌规范创建文档
- 使用组织特定的工作流分析数据
- 自动化个人任务
- 创意应用（艺术、音乐、设计）
- 技术任务（测试 Web 应用、生成 MCP 服务器）
- 企业工作流（沟通、品牌管理）

Skills 都能帮助 Claude 更专业、更高效地完成这些工作。

## Skills 的独特优势

1. **按需加载**：Skills 只在需要时加载，你可以维护数百个 Skills 而不影响性能
2. **跨平台兼容**：Claude Skills 已成为 Agent Skills 标准，被多个 AI 编程助手采用，包括 Codex、Gemini CLI、Cursor 等
3. **易于创建**：只需一个包含 `SKILL.md` 文件的文件夹即可

## 如何创建一个 Skill

创建 Skill 非常简单，只需要一个文件夹和一个 `SKILL.md` 文件：

```yaml
---
name: my-skill-name
description: 清晰描述这个 skill 的功能和使用场景
---

# My Skill Name

[在这里添加 Claude 激活此 skill 时需要遵循的指令]

## 示例
- 示例用法 1
- 示例用法 2

## 指南
- 指南 1
- 指南 2
```

**必需的 frontmatter 字段：**
- `name` - 唯一标识符（小写，用连字符代替空格）
- `description` - 完整描述 skill 的用途和使用场景

## 如何使用 Skills

### 在 Claude Code 中使用

```bash
# 通过插件市场添加
/plugin marketplace add anthropics/skills

# 然后浏览并安装
/plugin install document-skills@anthropic-agent-skills
/plugin install example-skills@anthropic-agent-skills
```

### 在 Claude.ai 中使用

所有示例 skills 对付费用户可用，也可以通过设置上传自定义 skills。

### 通过 Claude API 使用

参考 [Skills API Quickstart](https://docs.claude.com/en/api/skills-guide#creating-a-skill)
