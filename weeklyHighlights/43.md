## 1.1 新iArt.ai – AI动态图形生成工具，专注矢量动态图形

iArt.ai是AI驱动的动态图形生成工具，定位为Adobe After Effects的零学习曲线替代方案。用户通过自然语言、Figma设计或音频输入，可在30秒内生成广播级矢量动画，支持4K导出等功能，让无经验者也能快速制作专业动态图形。

#### 1.1.1 核心功能
1. 自然语言生成：输入创意描述，AI自动完成布局、动画等全流程，30秒输出专业动态图形。
2. 多格式导入支持：支持上传Figma设计文件或输入音频，基于素材生成匹配的动态效果。
3. 专业级导出：支持4K@60fps一键导出，满足广播级、演示级的画质需求。
4. 即时迭代修改：通过秒级重提示功能，可快速调整创意，实现动态图形的即时优化。
5. 多语言全流程支持：原生支持多语言提示词与生成，无需手动处理语言适配。

#### 1.1.2 技术原理
基于AI代理架构，搭载自然语言处理（NLP）模型理解用户创意需求，结合计算机视觉技术解析Figma设计或音频节奏。通过预训练的动画生成模型，自动完成关键帧计算、元素布局与转场效果，采用矢量图形渲染引擎保证画面无损缩放，同时集成MCP工具链实现多格式输出与第三方工具对接。

#### 1.1.3 应用场景
1. 职场演示：职场人士输入演示主题，快速生成Keynote级动态图形，用于汇报PPT提升视觉效果。
2. 短视频制作：自媒体创作者通过描述创意，30秒生成产品解释、知识科普类动态短视频，降低制作成本。
3. 设计原型动效：UI设计师上传Figma原型，自动生成交互动效，直观展示产品交互逻辑。

#### 1.2.1 核心功能

HPC-Ops是腾讯混元AI Infra团队开源的工业级高性能大模型推理算子库，覆盖Attention、MoE、GEMM等核心模块，原生支持BF16/FP8混合精度。该库针对NVIDIA H20等GPU深度优化，通过动态调度与算子融合使端到端QPM提升30%+，多项性能超越vLLM、FlashInfer等主流基线，已支撑腾讯混元大规模生产级推理服务。

#### 2.5.3 应用场景
1. 大规模LLM推理服务：适用于云厂商与AI服务商的生产级大模型部署，通过动态调度与算子融合提升服务吞吐量，降低单Token推理成本，已支撑腾讯混元大模型的日常服务。

4. 内置可编程浏览器：在终端旁可打开浏览器窗格，支持抓取页面无障碍树快照、执行点击填表及JS代码，让AI智能体直接操作本地开发服务器。
5. SSH远程工作区管理：为远程机器创建独立工作区，浏览器窗格通过远程网络路由使localhost直接可用，图片拖拽进远程会话可自动通过scp上传。
6. CLI与Socket自动化接口：提供命令行接口和Unix Socket API，支持脚本化创建工作区、分屏、标签页，实现AI编程工作流的全自动化编排。

#### 2.13.2 技术原理
1. 原生架构与渲染：底层采用libghostty终端渲染库，继承其GPU加速渲染能力；上层GUI基于Swift+AppKit原生开发，规避Electron/Tauri的性能与内存开销，终端键位绑定直接复用Ghostty配置文件。
2. 通知捕获与分发：通过标准终端转义序列自动监听AI智能体进程状态，也可通过CLI和Claude Code hooks手动触发；当智能体需介入时，在目标面板绘制蓝色通知环，同步更新侧边栏标记、弹出通知面板并推送原生桌面通知。
3. 可编程控制架构：所有工作区、分屏、标签页、浏览器及输入操作均通过Unix Socket API暴露，外部进程或脚本可直接连接控制终端布局与内容流；同时提供CLI封装，支持AI智能体自身或CI流水线自动化编排多会话任务。
4. 会话快照与恢复：退出时保存会话快照至`~/Library/Application Support/cmux/`，AI智能体钩子将会话映射存储在`~/.cmuxterm/`；重启时重建布局，对支持的智能体自动执行恢复命令，也支持自定义恢复配置。

#### 2.13.3 应用场景
1. 多智能体并行开发：面向全栈开发者，同时运行4-6个Claude Code/Codex会话处理不同模块，通过垂直标签页实时追踪各任务进度，提升多任务并行开发效率。
2. 全栈本地调试：后端开发者可在终端运行服务，同窗口以浏览器分屏访问localhost，让AI智能体直接抓取页面DOM执行前端测试，实现前后端一体化调试。
3. 远程服务器运维：运维人员通过SSH创建远程工作区，在浏览器直接访问远程内网服务，拖拽上传截图或日志文件，无需手动输入scp命令，简化远程操作流程。
4. 自动化工作流编排：DevOps工程师通过Socket API脚本定时创建标准分屏布局，自动分发任务到各AI智能体面板，实现代码审查、自动化测试等流程的无人值守运行。

## 2.14 BigSet – TinyFish 开源的多智能体实时网络抓取工具

BigSet是美国初创公司TinyFish推出的开源多智能体网络抓取工具，采用AGPL-3.0协议。用户通过自然语言描述需求，系统可自动完成表结构推断、多源数据抓取、去重验证，最终生成CSV/XLSX格式的结构化数据集，还支持定时刷新保持数据实时性，无需手动编写爬虫或指定URL。

#### 2.14.1 核心功能
1. 自然语言建表：输入一句话描述数据需求，AI自动推断列名、数据类型和主键，无需手动设计表结构。
2. 多智能体采集：编排器智能体发现目标实体，子智能体并行抓取数据，自动完成从发现到填充的全流程。
3. 自动去重溯源：基于主键自动去重，每行数据附带来源URL，确保数据可追溯验证。
4. 定时刷新：支持30分钟、6小时、12小时、每日、每周五种频率自动更新，保持数据集实时同步。
5. 格式导出：生成的结果支持CSV和XLSX格式下载，方便接入Excel或数据分析工具。
6. 预置数据集：内置9个精选公开数据集，用户可直接开箱即用。

#### 2.14.2 技术原理
基于Mastra框架构建多智能体系统，核心由编排器和子智能体组成，通过Vercel AI SDK与OpenRouter调度Claude Sonnet和Qwen模型。接收自然语言需求后，由Claude Sonnet分析语义并自动推断表结构Schema；编排器将目标拆分为任务分派给子智能体，子智能体调用TinyFish Search/Fetch/Browser API抓取数据，系统基于主键去重并附加来源URL。采用JS闭包注入数据集ID，防止提示词注入攻击，实现采集逻辑与数据权限隔离。技术栈包含Next.js 16、Fastify、Convex数据库等，支持Docker一键部署和本地开发。

#### 2.14.3 应用场景
1. 竞品监控：企业市场人员可通过自然语言需求，自动抓取竞争对手的产品价格、功能更新、招聘信息等，生成定时刷新的情报表。
2. 市场研究：行业分析师可收集特定行业的公司信息、融资动态、市场份额数据，快速构建行业数据库辅助研究。
3. 投资分析：投资者可跟踪股票、加密货币、初创公司的融资轮次及估值变化，获取支持决策的结构化数据。
4. 招聘情报：HR或求职者可监控目标公司或行业的职位发布、技能要求、薪资范围，分析人才市场趋势。
5. 电商比价：电商从业者或消费者可抓取多平台商品的价格、库存、评价数据，建立实时更新的比价数据集。

* GitHub仓库：https://github.com/tinyfish-io/bigset

## 2.15 SciDraw AI – AI 科学绘图平台，文字描述生成科研配图

SciDraw AI是面向科研人员、研究生和教师的AI驱动科学绘图平台，支持通过文字描述、草图输入秒级生成多学科科研配图，涵盖论文配图、技术路线图等场景，支持SVG矢量导出，可直接用于期刊投稿，大幅降低科研绘图的专业门槛。

#### 2.15.1 核心功能
1. **文生图功能**：输入自然语言描述，AI自动生成分子结构、信号通路、实验流程图等专业学术插图，满足科研可视化需求。
2. **草图精修功能**：上传手绘草图，AI将其转化为符合期刊标准的精细矢量图，保留原始构图逻辑，提升绘图专业度。
3. **风格复刻与编辑功能**：上传参考图可复刻其风格，还能通过多轮对话持续优化插图细节，适配不同场景的视觉要求。
4. **数据可视化功能**：上传CSV/Excel文件，AI自动推荐图表类型，生成箱线图、热图等，并附带Matplotlib/Seaborn代码，实现数据与插图一体化。
5. **期刊风格预设功能**：内置Nature、Science等顶刊配色与排版模板，一键适配投稿要求，降低格式调整成本。

#### 2.15.2 技术原理
基于大语言模型与多模态生成架构，文生图功能通过文本编码模型将自然语言转化为语义向量，输入扩散模型生成科研领域专属可视化图谱；草图精修采用CV领域的图像分割与风格迁移算法，保留草图结构信息的同时替换为期刊级视觉元素；数据可视化模块集成统计分析引擎，自动识别数据特征并调用Matplotlib/Seaborn渲染库生成图表，全流程通过Prompt工程实现多轮交互优化，确保输出结果符合科研领域专业规范。

#### 2.15.3 应用场景
1. **期刊论文投稿**：科研人员使用文生图功能快速生成图形摘要、机制图，搭配期刊风格预设和投稿预检器，直接满足顶刊投稿格式要求。
2. **毕业论文创作**：研究生通过草图精修功能将手绘的实验流程图转化为标准矢量图，统一全文插图风格，提升论文视觉专业性。
3. **基金申请材料制作**：科研工作者利用专用工具页生成技术路线图、研究框架图，清晰呈现研究逻辑，提升基金申请通过率。
4. **教学内容设计**：教师借助平台生成直观的科学示意图，用于课堂课件、科普文章，帮助学生理解复杂科学概念。
